# Acerca de esta web

Esta página es una **wiki temática** dedicada a *Hunter x Hunter*.  
Su objetivo es ofrecer un resumen organizado de su mundo, personajes, poder Nen y arcos principales.

### ¿Para quién es esta web?
- Estudiantes que preparan un proyecto temático.  
- Fans que quieren una visión clara y resumida.  
- Personas nuevas en Hunter x Hunter que quieran entender su universo.

### Contenido y fuentes
El contenido está redactado con fines educativos y divulgativos.
